# 수험생 학습 습관 자가진단 체크리스트

## 프로젝트 개요

수험생이 본인의 학습 습관을 82문항으로 자가진단하고, MBTI 유형 및 카테고리별 맞춤 솔루션을 제공하는 정적 웹 앱입니다. 책 『오직 나를 위한 공부』의 내용을 기반으로 합니다.

---

## ✅ 완료된 기능

### 인트로 화면 (`index.html`)
- **MBTI 선택**: 16가지 MBTI (분석가/외교관/관리자/탐험가형 4그룹) 선택
- **시험 유형 선택**: 객관식 / 논술형·주관식
- **시험 종류 선택**: 변리사, 세무사, 공무원 등 29개 카테고리
- 3가지 모두 선택 시 시작 버튼 활성화

### 설문 화면
- 82문항 / 11개 카테고리 (인풋 학습, 객관식, 강의 수강, 아웃풋·정리, 공부 전략, 시험·모의고사, 교재·자료, 복습·단권화, 계획 관리, 컨디션·수면, 멘탈 관리)
- 카테고리 탭 (스크롤 연동 + 진행 표시)
- 진행 바 / 답변 카운터
- 객관식 시험 해당 항목 → '해당없음' 선택지 표시
- 미답변 항목 강조(shake) 경고

### 결과 화면 ✨ (최신 개편)
- **요약 스코어** : 개선 필요 / 잘 모르겠다 / 양호 3개 카드
- **MBTI 맞춤 조언** : 선택한 MBTI 유형별 강점·약점 분석 + 3개 구체적 행동 조언
- **카테고리별 진단 & 솔루션 (11개)**
  - 위험도 퍼센트 바 + 위험/주의/양호 배지
  - 개선 필요 항목 목록 (빨간 배경)
  - 확인 필요 항목 목록 (노란 배경)
  - 카테고리별 4개 솔루션 팁 카드
  - 위험/주의 카테고리는 자동 펼침, 양호는 접힘 (클릭으로 토글)
- 다시 진단하기 / 전체 통계 보기 버튼

### DB 저장 (`tables/checklist_responses`)
- 응답 저장: session_id, exam_type, exam_category, mbti, submitted_at (ISO 문자열), q1~q82
- 저장 실패 시 결과 화면은 그대로 표시 (오프라인 폴백)

### 전체 통계 (`stats.html`)
- 응답 집계 / 카테고리별 위험도 순위 / 문항별 상세 통계
- 시험 유형·종류·MBTI 필터


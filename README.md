# 수험생 학습 습관 자가진단 체크리스트

## 프로젝트 개요
수험생들이 자신의 학습 습관을 진단하고 개선점을 파악할 수 있는 인터랙티브 설문 페이지입니다.  
총 82문항에 대해 **맞다 / 잘 모르겠다 / 아니다** 로 응답하며, 결과는 데이터베이스에 저장되어 전체 통계로 제공됩니다.

---

## 완성된 기능

### 설문 페이지 (`index.html`)
- ✅ 인트로 화면 (시작 안내)
- ✅ 82문항 전체 표시 (11개 카테고리 섹션 구분)
- ✅ 각 문항별 선택 버튼 (맞다 / 잘 모르겠다 / 아니다 / 해당없음*)
  - *객관식 시험 관련 항목에만 '해당없음' 옵션 추가
- ✅ 상단 고정 카테고리 탭 (클릭 시 해당 섹션으로 이동)
- ✅ 실시간 진행률 표시바
- ✅ 미응답 항목 표시 및 경고
- ✅ 결과 화면 (카테고리별 분석, 개선 필요 항목 리스트)
- ✅ 응답 데이터 DB 저장

### 통계 페이지 (`stats.html`)
- ✅ 전체 응답자 수, 평균 위험 항목 수 등 요약 카드
- ✅ 전체 응답 분포 도넛 차트
- ✅ 카테고리별 위험도 레이더 차트
- ✅ 카테고리별 상세 분석 카드
- ✅ 가장 많이 체크된 위험 항목 TOP 15 랭킹
- ✅ 항목별 상세 통계 (필터 및 정렬 기능)
